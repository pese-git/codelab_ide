# Example Project

This is a sample project to demonstrate CodeLab IDE features.

## Files Included

- `hello.dart` - Dart example with classes and functions
- `example.py` - Python example with similar functionality
- `README.md` - This documentation file

## Features Demonstrated

1. **File Tree Navigation** - Browse project structure
2. **Code Editor** - Edit files with syntax highlighting
3. **Terminal** - Run commands and scripts
4. **Multi-language Support** - Support for Dart, Python, and other languages

## How to Use

1. Open this project in CodeLab IDE
2. Click on files in the file tree to open them
3. Edit code in the editor
4. Use the terminal to run commands like:
   - `dart run hello.dart`
   - `python example.py`
   - `ls -la` (list files)
